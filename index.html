Projecten

Natours

❮ ❯

Natours is een project dat ik gebouwd heb met behulp van de Udemy course: Node.js, Express, MongoDB & More: The Complete Bootcamp 2023. Natours is een webapplicatie waar gebruikers toeren kunnen boeken. De Natours API is volgens het REST model gemaakt. Gebruikers kunnen registreren en inloggen, toeren bekijken/boeken, reviews schrijven over de toeren en hun profiel bewerken. De authenticatie wordt gedaan met JWT tokens. De frontend van de applicatie is server side rendered met behulp van pug templates en de backend is gemaakt in nodejs met express en voor de database mongodb.

Japero

❮ ❯

Voor western manege Japero in Friesland Berlikum heb ik een website gemaakt. Deze website is gemaakt met Wordpress zodat de manege zelf hun teksten en foto’s gemakkelijk kunnen aanpassen.
